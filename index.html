Safety Penguin is a proprietary web application which allows users to passively stay up-to-date on the wellfare of loved ones. Using a web or SMS text message interface, users can schedule check-in times. If they don't check in with Safety Penguin before that time, their chosen loved one will receive a notification reminding them to check on the user.
